Sistema de Gerenciamento de Campus

Este projeto foi desenvolvido com o objetivo de praticar Programação Orientada a Objetos (POO) em Python, evoluindo desde conceitos básicos até uma estrutura mais completa, envolvendo múltiplas classes, relacionamentos e organização de módulos.

O sistema simula um ambiente universitário simples, permitindo o cadastro e gerenciamento de campus, cursos e turmas, oferecendo uma visão geral de como aplicar os pilares de POO em um caso realista.

Objetivos do Projeto

Praticar a criação e uso de classes.

Entender como funciona herança, encapsulamento e composição.

Organizar um projeto em múltiplos módulos.

Criar métodos de listagem e cadastro em diferentes níveis.

Reforçar a lógica de relacionamento entre objetos.
Funcionalidades

O sistema permite:

1. Cadastrar campus

Cada campus possui nome, cidade e seus próprios cursos e turmas.

2. Cadastrar cursos

Um curso contém nome, coordenador e outros atributos simples.

3. Cadastrar turmas

As turmas são associadas a um curso específico dentro de um campus.

4. Listar informações

Listar cursos de um campus.

Listar turmas de um campus.

Listar todos os campus registrados.
